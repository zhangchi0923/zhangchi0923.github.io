如何搭建本站

May 12, 2019
1 min read

1.部署环境

  • macOS（真方便）
  • git（macOS系统自带，真方便）
  • brew macOS软件安装工具（https://brew.sh/index_zh-cn）官网直接复制命令即可下载安装
  • 安装hugo
安装brew

看见下图就算是安装成功了

安装hugo

用下面的命令即可安装

brew install hugo

看见下面返回值就算是安装成功了

可用下面命令查看版本

hugo version

2.使用hugo搭建个人博客

  • 新建博客文章
  • 部署网站至远端
新建博客文章

hugo new site yourblogname

事实上，本地的静态博客已经搭建好了（真方便）。

cd /Users/username/yourblogname
